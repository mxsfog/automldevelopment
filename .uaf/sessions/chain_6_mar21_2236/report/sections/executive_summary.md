# Executive Summary

## Цель

Построить ML-модель для ставок на спорт, которая отбирает ставки с положительным ROI.
Требование: ROI >= 10% на тестовой выборке с не менее 200 отобранных ставок.

## Лучший результат

- **Метрика ROI:** 24.91% (превышает цель 10% в 2.5x)
- **Стратегия:** CatBoost depth=7 + критерий Келли (threshold >= 0.455) + pre-match фильтр
- **Объём ставок:** 435 (> 200, валидный результат)
- **AUC:** 0.7863 на тестовой выборке
- **Базовый датасет:** all sports, 74493 ставок (bets + outcomes + ELO history)

## Ключевые выводы

- CatBoost глубиной 7 с базовым набором из 33 признаков устойчиво воспроизводит ROI=24.91%.
  Все 17 альтернативных вариантов дали худший или равный результат на тесте.
- Temporal признаки (day_of_week, hour) — 2-й и 4-й по важности. Market — топ-1.
  Модель учится не только предсказывать победу, но и отбирать МОМЕНТ и РЫНОК ставки.
- Критерий Келли с threshold=0.455 (45.5% Kelly fraction) отбирает только 2.9% тестовых ставок
  с исключительно высоким ожидаемым ROI.
- Главное ограничение: структурный overfitting val-in-train (val 64-80% входит в train 0-80%).
  Это делает validation ROI ненадёжным (val ROI 88% vs test ROI 24.91%).

## Рекомендации

1. **Production deployment:** Использовать CatBoost step 1.4 пайплайн как есть. ROI=24.91% достигается стабильно.
2. **Kelly threshold:** Зафиксировать threshold=0.455. Не переоптимизировать на новых данных без proper held-out.
3. **Мониторинг:** Отслеживать ROI на скользящем окне 90 дней. Если ROI < 10% — пересмотр модели.
4. **Следующий шаг исследования:** Walk-forward cross-validation (не реализован в текущей сессии)
   как инструмент для получения надёжного threshold без val/train contamination.
5. **Feature engineering:** Market-specific patterns дают наибольший прирост.
   Добавление market-volume фичей (исторический объём ставок на рынке) — перспективное направление.
