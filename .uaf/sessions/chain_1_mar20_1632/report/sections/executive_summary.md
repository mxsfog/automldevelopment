# Executive Summary

## Цель
Построить ML-модель предсказания исходов ставок (won/lost) на спортивных событиях для отбора ставок с ROI >= 10%. Данные: 74 493 ставки (синглы и парлаи) по 10+ видам спорта, период декабрь 2025 -- февраль 2026.

## Лучший результат
- Метрика ROI: **7.26%** (val-selected threshold, anti-leakage compliant)
- Стратегия: Ensemble (CatBoost + LightGBM + XGBoost), средние вероятности, порог 0.60, исключение убыточных спортов (Basketball, MMA, FIFA, Snooker)
- AUC: 0.8097
- Объем ставок: 5 146 из 12 118 (42.5% отфильтрованного test set)
- Win rate: ~76%

## Ключевые выводы
- Segment filtering (исключение 4 убыточных спортов) дал наибольший прирост ROI: с 3.20% до 7.23% (+4.03 п.п.)
- Odds/implied_prob доминируют в feature importance (85%+). Модель преимущественно обучается на структуре коэффициентов
- Потолок ROI при текущих данных: ~7.2%, подтверждено 8 итерациями Phase 4 без существенного улучшения
- Target 10% не достигнут. Разрыв ~2.7 п.п. требует принципиально новых данных или подходов

## Рекомендации
1. **Обогащение данных**: добавить внешние данные (ELO, form, head-to-head) -- текущие ML_Team_Stats_Found всегда 'f'
2. **Расширение временного окна**: 3 дня test set -- слишком мало для надежной оценки стабильности
3. **Динамическое исключение спортов**: автоматический мониторинг ROI по спортам с rolling window
4. **Online learning**: адаптация порога и весов ансамбля на свежих данных
