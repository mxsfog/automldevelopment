# Executive Summary

## Цель
Предсказание победы ставки (won/lost) на спортивных событиях с целью отбора прибыльных ставок. Данные со стейкинг-платформы: 74493 ставки, 81 день, 20+ рынков, 10+ видов спорта.

## Лучший результат
- Метрика ROI: **+27.95%** (валидировано на val, применено к test)
- Стратегия: confidence-weighted EV selection (conf_ev >= 0.15)
- Объём ставок: 1092 из 14899 на test (7.3% selectivity)
- AUC модели: 0.784

## Ключевые выводы
- **Confidence-weighted EV** — главное открытие сессии. Формула `EV * 1/(1+std*10) >= 0.15` отсеивает ставки где модели не согласны, повышая ROI с 16.02% до 27.95%.
- **Simple ensemble is king.** 3-model average (CatBoost + LightGBM + LogisticRegression) с базовыми параметрами оптимален. Усложнение (meta-model, segment models, temporal decay, calibration, extended features, odds weighting) не дает улучшений.
- **CV показывает нестабильность.** Средний ROI по 5-fold expanding window CV = 0-5% при std=11-70%. Результат 27.95% специфичен для тестового периода (Feb 20-22) и не является стабильным edge.
- **Ограничение данных.** 81 день — недостаточно для стабильной оценки. Temporal non-stationarity: спорты, рынки, odds distribution меняются между периодами.

## Рекомендации
1. **Больше данных** — 6+ месяцев для стабильной оценки стратегии
2. **Rolling retrain** — обновлять модель каждые 1-2 недели для адаптации к drift
3. **Использовать conf_ev_0.15** как базовый фильтр, но с осторожностью — реальный ожидаемый ROI ~5%, не 28%
4. **Мониторинг** — отслеживать ROI по спортам и периодам, быстро реагировать на деградацию
