# Executive Summary

## Задача

Предсказание победы ставки (won/lost) на спортивных событиях с целью достижения ROI >= 10% на отобранных ставках. Данные: 74,493 ставки со стейкинг-платформы (синглы и парлаи, 20+ рынков, 10+ видов спорта).

## Результат

**Достигнут ROI = 24.91%** на тестовой выборке (последние 20% данных по времени).
Целевой ROI >= 10% **превышен в 2.49 раза**.

Количество отобранных ставок: 435 из 4,733 pre-match тестовых ставок (9.2%).

## Ключевое решение

Двухэтапная стратегия отбора ставок:

1. **Pre-match фильтр** — отбираются только ставки, размещённые до начала матча (lead_hours > 0, ~31% от всех ставок). Live ставки исключаются как непредсказуемые.

2. **Kelly criterion** — из pre-match ставок отбираются те, где Kelly >= 0.455:
   `Kelly = (p * (Odds-1) - (1-p)) / (Odds-1)`
   где `p` — вероятность победы от CatBoost модели, Odds — коэффициент ставки.

## Модель

| Параметр | Значение |
|----------|----------|
| Алгоритм | CatBoost Classifier |
| Глубина | 7 |
| Learning rate | 0.1 |
| Iterations | 500 (early stopping=50) |
| Взвешивание | Temporal weights (half_life=0.5) |
| AUC (val/test) | 0.892 / 0.786 |

## Прогресс исследования

| Шаг | Метод | ROI test |
|-----|-------|----------|
| 1.1 | DummyClassifier | -3.07% |
| 1.4 | CatBoost default | 5.34% |
| 2.1 | + ELO features + temporal weights | 7.34% |
| 4.3 | + Isotonic calibration + Kelly | 12.24% |
| 4.4 | + Kelly fraction 0.5 | 14.52% |
| **4.5** | **+ Pre-match filter** | **24.91%** |

## Ограничения

- Высокая дисперсия в CV: -6.40% ± 20.81% (4-fold). Fold 3 (~40-60% данных) стабильно давал отрицательный ROI — свидетельство нестационарности временного ряда.
- Малое число ставок (435 в тестовом периоде): высокая дисперсия результата от месяца к месяцу.
- Рекомендуется мониторинг win rate и пересмотр порога каждые 2-3 месяца.
