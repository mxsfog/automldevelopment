# Executive Summary

## Цель

Разработать систему отбора ставок на спорт с ROI >= 10% на тестовом наборе данных. Использовались исторические одиночные и комбинированные ставки со стейкинг-платформы (20+ рынков, 10+ видов спорта).

## Лучший результат

- **Метрика ROI:** 28.5833%
- **Стратегия:** CatBoost (глобальная модель, chain_6_mar21_2236) + фильтр рынка 1x2 + shrunken segment Kelly thresholds (shrinkage=0.5)
- **Объём ставок:** 233 ставки из 14 899 тестовых (1.6%)
- **AUC модели:** 0.7863
- **Порог pre-match фильтра:** lead_hours > 0

## Ключевые выводы

- CatBoost с категориальными признаками (Sport, Market, Currency) обеспечивает наилучшую калибровку вероятностей для Kelly criterion. Альтернативы (LightGBM, XGBoost) дают значительно худший ROI.
- Рынок 1x2 (футбольный матч-результат) является доминирующим прибыльным сегментом: все 233 отобранных ставки относятся к Soccer/1x2.
- Shrunken segment thresholds (регуляризация порогов по Kelly-бакетам на 50% к baseline) улучшают результат относительно единого глобального порога (+3.67% vs step 4.6).
- Обучение специализированной модели только на 1x2-данных (step 4.11) приводит к переобучению (7150 примеров — слишком мало), глобальная модель с фильтрацией работает лучше.
- Структурное ограничение: val-set перекрывается с train (64-80% vs 0-80%), вызывая искусственно высокий val ROI (~88-200%). Честное разбиение (step 3.3) даёт только 0.94% ROI — ceiling системы находится в ~25-30%.

## Рекомендации

1. **Производство:** развернуть фильтр 1x2 + shrunken segments как live-стратегию с Kelly fraction 0.5 от рекомендуемой ставки для управления дисперсией.
2. **Следующая итерация:** собрать 3-5x больше 1x2 данных для обучения специализированной модели (step 4.11 показал, что при достаточном объёме данных такой подход перспективен).
3. **Мониторинг:** отслеживать rolling 30-day ROI; при падении ниже 5% — переобучить модель на последних данных.
4. **Расширение рынков:** проверить Asian Handicap как второй прибыльный рынок (показал хорошие val-метрики в step 4.9).
