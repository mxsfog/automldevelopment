\section{Executive Summary}

\subsection{Executive Summary}

\subsection{Цель}
Предсказание победы ставок на спортивных событиях для достижения ROI >= 10\% на отобранных ставках. Данные со стейкинг-платформы: синглы и парлаи, 20+ рынков, 10+ видов спорта.

\subsection{Лучший результат}
\begin{itemize}
  \item Метрика ROI: \textbf{28.44\%} (test), \textbf{22.42\%} (5-fold CV avg)
  \item Стратегия: CatBoost (depth=8, lr=0.08, l2=21.1), full-train на ELO-подмножестве с sport filter (без Basketball, MMA, FIFA, Snooker), выбор ставок: p>=0.77 AND EV>=0 (p*odds>=1)
  \item AUC: 0.8623
  \item Объём ставок: 328 из 1094 sport-filtered ELO test bets (30\% coverage)
\end{itemize}

\subsection{Альтернатива (без EV фильтра)}
\begin{itemize}
  \item ROI: 21.31\% (test), 11.02\% (CV avg)
  \item Объём: 463 ставок (42\% coverage)
\end{itemize}

\subsection{Ключевые выводы}
\begin{itemize}
  \item EV фильтр (p*odds>=1) -- главное открытие chain\_4, даёт +7.1 п.п. на test, +11.4 п.п. на CV
  \item ELO-фичи (winrate, elo\_diff, team stats) -- главный предиктор качества ставки
  \item Full-train модель (100\% train data) даёт +1 п.п. vs 80/20 split
  \item CatBoost solo доминирует все альтернативы (LGB, blends, stacking, ensembles)
  \item Sport filter стабильно добавляет 2-5 п.п., SF подход 5/5 positive folds vs ELO\_all 4/5
  \item Порог p=0.77 подтверждён val-sweep (step 4.14): ROI стабилен в диапазоне 0.75-0.78
  \item Odds-range analysis: ставки с odds 1.01-1.15 дают только 0.95\% ROI, EV фильтр их удаляет
\end{itemize}

\subsection{Рекомендации}
\begin{enumerate}
  \item Использовать CatBoost full-train + EV>=0 + p>=0.77 как основную стратегию
  \item Fallback: p>=0.77 без EV фильтра для максимального покрытия ставок (21.31\% ROI)
  \item Расширить ELO-трекинг: текущее покрытие 9.7\% ограничивает объём
  \item Rolling window мониторинг ROI по спортам с алертами при деградации
  \item Тестирование на 2+ недельном out-of-time окне перед production
  \item Dual-model: ELO-SF модель для ELO ставок, fallback модель для остальных
\end{enumerate}

\subsection{Baseline Performance}
\begin{itemize}

\subsection{Feature Engineering Results}
\subsubsection{Что улучшило модель}
\begin{itemize}

\subsection{EV Filter Discovery (step 4.9-4.11)}
\subsubsection{Механизм}
EV = predicted\_prob * odds - 1. Фильтр EV>=0 требует, чтобы ожидаемая ценность ставки была неотрицательной по оценке модели.

\subsubsection{Что удаляет EV фильтр}
\begin{itemize}
  \item 135 ставок из 463 (при p>=0.77)
  \item Удалённые: avg odds = 1.05 (очень низкие коэффициенты)
  \item Оставленные: avg odds = 1.32
  \item Soccer: -7.45\% ROI среди удалённых
  \item Table Tennis: -8.10\% ROI среди удалённых
  \item Фильтр убирает ставки на heavy favorites с минимальной маржей
\end{itemize}

\subsection{Threshold Sensitivity (step 4.14)}
Val-sweep и test-sweep подтверждают p=0.77 как оптимальный порог:

ROI плавно снижается в обе стороны от 0.76-0.77 -- подтверждает робастность выбора.

\subsection{Stability \& Validity}
\subsubsection{Anti-leakage проверки}
\begin{itemize}
  \item Threshold (p>=0.77) определяется на val (last 20\% of train)
  \item EV>=0 -- fixed domain-knowledge rule, не подбирается на test
  \item ELO фичи используют Old\_ELO (до матча), нет future leakage
  \item Time-series split: train до 2026-02-20, test после
  \item Final ROI (28.44\%) < 30\% -- в пределах допустимого
  \item SF approach validated: 5/5 positive folds in CV
\end{itemize}
