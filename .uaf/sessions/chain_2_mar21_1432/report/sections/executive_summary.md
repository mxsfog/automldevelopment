# Executive Summary

## Цель
Продолжение исследования (chain continuation) предсказания победы ставок на спортивных событиях. Цель — улучшить ROI >= 10% на отобранных ставках поверх baseline 16.02% из предыдущей сессии (chain_1_mar21_1356).

## Лучший результат
- Метрика ROI: **+16.02%** (не улучшен)
- Стратегия: 3-model ensemble (CatBoost + LightGBM + LogReg) на полном train, отбор по Expected Value >= 0.12
- Объём ставок: 2247 из 14899 тестовых (15.1%)
- AUC: 0.784, win rate: 36.6%, avg odds: 33.9

## Ключевые выводы
- Baseline ensemble с простыми параметрами (depth=6, iterations=200) устойчив к любым попыткам улучшения
- Калибровка вероятностей (Platt/isotonic), более сложные модели (Optuna params, XGBoost), Kelly criterion, regression на profit — всё ухудшает результат
- Прибыль генерируется на high-odds value bets, где модель находит расхождение между предсказанной и implied вероятностью
- Главное ограничение: высокая дисперсия (CV std=14.14%), 1 из 5 фолдов на нуле

## Рекомендации
1. Зафиксировать текущую стратегию как production-ready (ROI=16.02% стабилен)
2. Расширить временное окно данных для снижения дисперсии
3. Исследовать agreement-based selection (не завершено из-за budget stop) — потенциально может снизить variance
4. Мониторинг drift: переобучать при деградации ROI ниже 5% на rolling window
