# Executive Summary

## Цель
Предсказание исхода спортивных ставок (won/lost) с целью отбора прибыльных ставок через EV-фильтрацию. Целевая метрика: ROI >= 10% на отобранных ставках.

## Лучший результат
- Метрика ROI: 65.76% (best seed), 61.30% avg (10 seeds)
- Bootstrap 95% CI: [47.16%, 84.53%], median=66.62%
- Стратегия: PS_floor15 — per-sport EV thresholds с floor=0.15, min_prob=0.77
- Модель: CatBoost (optimized params из chain_5), 33 фичи (base+engineered+ELO)
- Фильтры: has_elo==1.0, исключены Basketball/MMA/FIFA/Snooker
- AUC: 0.8662 avg (10 seeds), std=0.0026
- Объём ставок: 75-93 (зависит от seed)

## Ключевые выводы
- ELO-фичи и фильтрация по видам спорта — главные драйверы ROI (от 0% до 49%)
- Per-sport EV thresholds на мелкой сетке (0.005) дают дополнительные +5-13pp vs единый порог
- Модель стабильна: 10 seeds дают ROI от 55.75% до 65.76%, std=2.92%
- Нижняя граница 95% CI (47.16%) значительно выше целевых 10%
- Temporal features и stacking ансамбли не улучшают результат на данном объёме данных
- Interaction features улучшают AUC (+0.01) но не ROI

## Рекомендации
- Использовать модель seed=456 как production baseline
- Переоценить per-sport EV thresholds при появлении новых данных
- Мониторить ROI по спортам: при деградации пересмотреть sport filter
- Исследовать калибровку вероятностей для дальнейшего улучшения EV-отбора
- Рассмотреть online learning для адаптации к сдвигу распределения odds
