\subsection{Лучший результат}
\begin{itemize}
  \item Метрика ROI: 65.76\% (best seed), 61.30\% avg (10 seeds)
  \item Bootstrap 95\% CI: [47.16\%, 84.53\%], median=66.62\%
  \item Стратегия: PS\_floor15 — per-sport EV thresholds с floor=0.15, min\_prob=0.77
  \item Модель: CatBoost (optimized params из chain\_5), 33 фичи (base+engineered+ELO)
  \item Фильтры: has\_elo==1.0, исключены Basketball/MMA/FIFA/Snooker
  \item AUC: 0.8662 avg (10 seeds), std=0.0026
  \item Объём ставок: 75-93 (зависит от seed)
\end{itemize}

\subsection{Ключевые выводы}
\begin{itemize}
  \item ELO-фичи и фильтрация по видам спорта — главные драйверы ROI (от 0\% до 49\%)
  \item Per-sport EV thresholds на мелкой сетке (0.005) дают дополнительные +5-13pp vs единый порог
  \item Модель стабильна: 10 seeds дают ROI от 55.75\% до 65.76\%, std=2.92\%
  \item Нижняя граница 95\% CI (47.16\%) значительно выше целевых 10\%
  \item Temporal features и stacking ансамбли не улучшают результат на данном объёме данных
  \item Interaction features улучшают AUC (+0.01) но не ROI
\end{itemize}

\subsection{Рекомендации}
\begin{itemize}

  \item Переоценить per-sport EV thresholds при появлении новых данных

\section{Analysis and Findings}

\subsection{Analysis and Findings}

\subsection{Baseline Performance}
\begin{itemize}
  \item DummyClassifier: ROI=-3.07\% — ставки без модели убыточны
  \item Rule-based (ML\_Edge>=11.5): ROI=-4.89\% — простые правила хуже случайного
  \item LogisticRegression: ROI=3.83\%, AUC=0.7947 — первый положительный ROI, линейная модель уже полезна
  \item CatBoost default: ROI=0.16\%, AUC=0.7981 — без оптимизации порогов CatBoost не лучше
\end{itemize}

Вывод: базовые модели без EV-фильтрации и feature engineering дают ROI около 0\%.

\subsection{Feature Engineering Results}

\subsubsection{Принятые фичи (33 total)}
\begin{itemize}
  \item \textbf{Base (8):} Odds, USD, ML\_P\_Model, ML\_P\_Implied, ML\_Edge, ML\_EV, Outcomes\_Count, Is\_Parlay\_bool
  \item \textbf{Engineered (10):} log\_odds, implied\_prob, value\_ratio, edge\_x\_ev, edge\_abs, ev\_positive, model\_implied\_diff, log\_usd, log\_usd\_per\_outcome, parlay\_complexity
  \item \textbf{ELO (15):} team\_elo\_mean, elo\_diff, team\_winrate\_diff, elo\_spread и др.
\end{itemize}

\subsubsection{Отклонённые фичи}
\begin{itemize}
  \item \textbf{Temporal (5):} hour, day\_of\_week, is\_weekend, cyclic encodings — AUC -0.005, ROI -8-12pp. Время события не несёт предиктивной информации.
  \item \textbf{Interactions (12):} wr\_diff\_x\_odds, odds\_x\_elo\_mean и др. — AUC +0.01, но ROI -1pp. CatBoost сам находит нужные interactions через splits.
\end{itemize}

\subsubsection{Ключевой инсайт}
ELO-фичи дают основной прирост: team\_winrate\_diff — top-1 по feature importance. Фильтр has\_elo==1.0 критичен: модель работает только на матчах с историей ELO.

\subsection{Model Comparison}
\begin{itemize}
  \item \textbf{CatBoost solo:} лучший результат. Optimized params из chain\_5 (depth=7, lr=0.05, l2=5.0).
  \item \textbf{Stacking (CB+LGB+XGB):} ROI=49.14\% — хуже solo CatBoost. Причина: данных мало (~5000 filtered samples), 3-way split уменьшает train слишком сильно.
  \item \textbf{LightGBM/XGBoost solo:} в stacking эксперименте показали AUC на уровне CatBoost, но ROI ниже из-за худшей калибровки.
  \item \textbf{Weighted average:} не улучшает vs CatBoost solo.
\end{itemize}

Вывод: на малых данных (5000 samples) одиночный CatBoost оптимален.

\subsection{Segment Analysis}

\subsubsection{Прибыльные спорты (после фильтрации)}
Модель обучается и предсказывает только на матчах с ELO, исключая Basketball, MMA, FIFA, Snooker. Оставшиеся спорты (Tennis, Football, Hockey, Baseball и др.) дают стабильно положительный ROI.

\subsubsection{Per-sport EV thresholds}
Разные спорты требуют разных EV порогов. PS\_floor15 стратегия подбирает оптимальный порог для каждого спорта на validation split с шагом 0.005, минимум floor=0.15. Это даёт +5-13pp ROI vs единый порог EV>=0.10.

\subsubsection{Odds диапазоны}
Odds range фильтры (1.1-5.0, 1.2-3.5 и т.д.) не улучшают ROI поверх per-sport EV thresholds.

\subsection{Stability and Validity}

\subsubsection{Multi-seed stability (10 seeds)}
\begin{itemize}
  \item ROI avg=61.30\%, std=2.92\%
  \item Все 10 seeds дают ROI > 55\%
  \item AUC avg=0.8662, std=0.0026
\end{itemize}

\subsubsection{5-fold expanding window CV}
\begin{itemize}
  \item PS\_floor15: avg=36.79\%, std=22.61\%, 4/5 folds positive
  \item PS\_floor10: avg=34.74\%, std=18.82\%, 5/5 folds positive
  \item CV ROI ниже test ROI — ожидаемо, т.к. ранние folds имеют меньше training data
\end{itemize}

\subsubsection{Bootstrap CI (1000 samples)}
\begin{itemize}
  \item 95\% CI: [47.16\%, 84.53\%]
  \item Median: 66.62\%
  \item Нижняя граница значительно выше 10\% target
\end{itemize}

\subsubsection{Anti-leakage проверки}
\begin{itemize}
  \item Thresholds подбираются только на val (последние 20\% train)
  \item Test используется один раз для финальной оценки
  \item Нет future leakage: все фичи доступны до события
  \item ELO-фичи рассчитаны на исторических данных (без lookahead)
\end{itemize}

\subsection{What Didn't Work}

\begin{enumerate}
  \item \textbf{Temporal features} (step 4.1): час, день недели, выходные — чистый шум. AUC и ROI падают.
  \item \textbf{Stacking ensemble} (step 4.2): CB+LGB+XGB с LogReg meta-learner. Данных слишком мало для 3-way split. Solo CatBoost лучше.
  \item \textbf{Interaction features} (step 4.5): 12 попарных комбинаций top фичей. AUC +0.01 но ROI не улучшился — CatBoost и так находит нужные splits.
  \item \textbf{Odds-stratified EV} (step 4.3): разные EV пороги для favorites/balanced/underdogs — хуже per-sport подхода.
  \item \textbf{EV+p sweep} (step 4.3): оптимизация единого EV и p порога — лучший единый порог EV=0.24+p=0.73 даёт ROI=62.80\%, но PS\_floor15 стабильнее на CV.
\end{enumerate}
