# Executive Summary

## Цель
Предсказание победы ставки (won/lost) на спортивных событиях. Цель: ROI >= 10% на отобранных ставках. Продолжение chain_1 (потолок 7.32% ROI).

## Лучший результат
- Метрика ROI: **18.61%** (robust multi-fold threshold t=0.73, leakage-free)
- Стратегия: Ensemble CB50 (Optuna CatBoost 50% + Optuna LightGBM 25% + XGBoost 25%) на **ELO-only subset** (ставки с данными ELO рейтингов команд)
- Объём ставок: 602 из 1332 ELO test (45.2% coverage), при t=0.40 ROI=9.07% на 1045 ставках (78%)
- AUC: 0.8471 на ELO test
- Робастный диапазон: 15-19% ROI в зависимости от метода выбора порога

## Ключевые выводы
- **ELO-данные -- главный прорыв**: chain_1 упёрся в 7.32% из-за доминирования Odds (85% FI). ELO-фичи (team_winrate_diff, elo_diff, team_winrate_mean) диверсифицировали feature importance и дали +9-10 п.п.
- **Все спорты прибыльны** на ELO-subset: Soccer 26%, Tennis 17%, Table Tennis 16%, Basketball 11%
- **Робастность подтверждена**: на 4 разных temporal splits mean ROI=12.15% (std=5.12%)
- **Ограничение**: ELO-данные доступны только для 9.7% ставок (7198/74493). Non-ELO ставки дают ROI ~2-5%

## Рекомендации
1. **Расширить ELO coverage**: увеличить количество матчей с ELO-трекингом. Каждый дополнительный % coverage -- потенциально +100-200 прибыльных ставок
2. **Мониторинг**: rolling window ROI по спортам. ELO-модель стабильна, но нужен дрифт-детектор
3. **Dual-model production**: использовать ELO-модель (ROI~16%) для ELO-ставок, chain_1 модель (ROI~7%) с segment filtering для остальных
4. **Расширить тест window**: текущий 3-дневный тест -- минимум. 2+ недели повысят достоверность
