\section{Analysis and Findings}

\subsection{Analysis and Findings}

\subsection{Phase 1: Baselines}
Установлены 4 базовые модели:
\begin{itemize}
  \item DummyClassifier: ROI=-3.07\% (lower bound)
  \item Rule ML\_Edge>=11.5: ROI=-4.89\% (простое правило хуже random)
  \item LogisticRegression: ROI=2.62\%, AUC=0.7943 (linear baseline, положительный ROI)
  \item CatBoost default: ROI=1.72\%, AUC=0.7930 (non-linear baseline)
\end{itemize}

\subsection{Phase 2: Feature Engineering}
Воспроизведен chain\_4 baseline: ROI=28.44\% (EV>=0+p77), AUC=0.8623.
ELO фичи + sport filter + full-train + EV фильтр дают основной скачок.

\subsection{Phase 3: Optuna}
Optuna overfitted (val=38\% -> test=25\%, delta=-3.54pp). Chain\_4 params подтверждены как оптимальные.

\subsection{Phase 4: Free Exploration}

\subsubsection{Ключевое открытие: строгость EV фильтра}
EV>=0.10 почти удваивает CV ROI по сравнению с EV>=0:
\begin{itemize}
  \item EV>=0: CV=11.10\%, test=28.44\%, n=134/fold
  \item EV>=0.10: CV=21.73\%, test=49.45\%, n=47/fold
\end{itemize}

Механизм: EV>=0.10 удаляет ставки с положительным, но маленьким EV (0-10\%).
Эти ставки имеют низкий ROI из-за маржи букмекера.

\subsubsection{Per-sport EV thresholds}
Per-sport оптимизация EV порога на val дает ещё +5.34pp по CV (27.07\% vs 21.73\%).
Разные спорта имеют разную оптимальную строгость:
\begin{itemize}
  \item Table Tennis, Tennis, Soccer: EV>=0.22-0.24 (нужен высокий edge)
  \item CS2, Dota 2: EV>=0.17
  \item Остальные: EV>=0.10 (floor)
\end{itemize}

\subsubsection{Что не сработало}
\begin{itemize}
  \item Time features (hour, day\_of\_week): -2.25pp
  \item Kelly criterion staking: хуже flat (44\% vs 49\%)
  \item LightGBM: CV=15.03\% vs CB=21.73\%
  \item CB+LGB blend: CV=24.23\% (лучше CB solo), но хуже CB+PS\_EV
  \item Blend+PS\_EV: CV=22.63\% (хуже CB+PS\_EV=27.07\%)
  \item Platt/Isotonic calibration: -2.9/-13.9pp
  \item Optuna re-tuning: overfitting
  \item Odds filter поверх EV>=0.10: не добавляет ценности
\end{itemize}

\subsubsection{Robustness}
\begin{itemize}
  \item Leave-one-sport-out: ROI=43-54\% при удалении любого спорта
  \item min\_bets sensitivity: стабильный (57.17-57.56\%)
  \item Все 5 фолдов CV положительные для всех ключевых стратегий
  \item Per-sport breakdown: Table Tennis 47.89\%, Tennis 38.86\%, Soccer 73.22\%, CS2 90.58\%
\end{itemize}
