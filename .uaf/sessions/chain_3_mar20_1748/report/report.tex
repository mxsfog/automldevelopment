\subsection{Executive Summary}

\subsection{Задача}
Предсказание исхода спортивных ставок (won/lost) для отбора прибыльных ставок с целью ROI >= 10\%.

\subsection{Результат}
\textbf{ROI = 20.23\%} на test-сете (3-дневное окно, 449 ставок из 1094 sport-filtered ELO подмножества).

\subsection{Стратегия}
\begin{itemize}
  \item \textbf{Модель:} CatBoost (depth=8, lr=0.08, l2\_leaf\_reg=21.1, min\_data\_in\_leaf=20)
  \item \textbf{Данные:} только ставки с ELO-рейтингами (~9.7\% от всех ставок)
  \item \textbf{Sport filter:} исключены Basketball, MMA, FIFA, Snooker из train и test
  \item \textbf{Порог:} фиксированный t=0.77 (выше порога -> ставка принимается)
  \item \textbf{Фичи:} 28 признаков (base odds + engineered + ELO features)
\end{itemize}

\subsection{Робастность}
4-fold temporal CV: mean ROI = 13.55\% +/- 8.09\%. Все 4 фолда положительные (min=3.72\%, max=25.13\%).

\begin{itemize}
  \item Без ELO: ROI=0.34\% (Odds-доминирование, 96\% FI)

\end{itemize}

ELO-фичи (team\_winrate\_diff, elo\_diff, k\_factor\_mean) диверсифицировали feature importance и дали модели информацию о силе команд, которую odds не полностью capture.
